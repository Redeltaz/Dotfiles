
# Mivoen

Setup :
```
git clone https://github.com/Redeltaz/mivoen ~/.config/nvim
```
<br>

Install language server :
<br>
```
npm i -g typescript \
         typescript-language-server \
         bash-language-server \
         dockerfile-language-server-nodejs \
         vim-language-server \
         vscode-langservers-extracted \
         pyright
```
